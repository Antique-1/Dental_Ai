# 2. 프로젝트 배경 및 필요성

현대 치의학에서는 환자의 구강 상태를 정확하게 진단하고 조기에 질환을 발견하는 것이 매우 중요합니다. 그러나 실제 임상 환경에서는 환자의 생활습관, 치아 상태, 잇몸 상태 등 다양한 요소가 복합적으로 작용하기 때문에 동일한 치료 방법을 적용하더라도 환자마다 치료 결과에 차이가 발생할 수 있습니다.

특히 충치(Caries), 치주질환(Periodontal Disease)과 같은 질환은 초기 발견 시 비교적 간단한 치료로 해결 가능하지만, 진단 시기가 늦어질 경우 치료 비용 증가 및 치아 손실로 이어질 수 있습니다.

최근 의료 분야에서는 머신러닝과 딥러닝 기술을 활용하여 의료 영상 및 환자 데이터를 자동 분석하는 연구가 활발하게 진행되고 있어, 치과 분야 역시 파노라마 X-Ray 영상과 환자 데이터를 기반으로 질환을 자동 탐지하고 진단을 보조하는 시스템에 대한 필요성이 증가하고 있습니다.

본 프로젝트에서는 치과 영상 데이터와 환자 데이터를 머신러닝 기반으로 분석하여 충치 여부 및 구강 상태를 자동 분류하고, 사용자 맞춤형 관리 방안을 제공하는 시스템을 구현하고자 합니다.

# 3. 프로젝트 목표

본 프로젝트의 주요 목표는 다음과 같다.

## 3.1 충치 및 구강 질환 자동 탐지
- 치과 X-Ray 이미지를 분석하여 충치 및 이상 부위를 자동 탐지
- 딥러닝 기반 객체 탐지(Object Detection) 모델 구현

## 3.2 환자 구강 상태 분류
- 환자의 치주 상태, 충치 여부, 생활습관 데이터를 기반으로 위험도 분류
- 머신러닝 알고리즘을 활용한 환자 유형 분석

## 3.3 사용자 맞춤형 관리 시스템 구축
- 분석 결과에 따라 환자별 관리 방법 제공
- 구강 상태에 맞는 예방 방법 및 사후관리 출력

## 3.4 웹 기반 서비스 구현
- Streamlit 기반 인터페이스 구현
- 사용자가 이미지를 업로드하면 분석 결과를 실시간으로 확인 가능하도록 개발

1. 이미지 데이터 전처리
2. YOLO 형식 라벨 변환
3. 데이터셋 분할(train / validation)

5. 성능 평가
