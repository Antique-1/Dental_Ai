## 2. 배경 및 필요성
- 환자마다 구강 상태 및 질환 진행 속도가 다름  
- 동일한 치료 방식 적용 시 효과 차이가 발생할 수 있음  
- 데이터 기반 분석을 통해 환자 유형을 분류하고 맞춤 치료 필요  

## 3. 목표
- 머신러닝을 활용하여 환자의 구강 데이터를 분석하고  
- 질환 위험도 또는 환자 유형을 자동 분류하는 모델 개발  

## 4. 주요 기능 및 특징
- 환자 데이터 분석  
  (예: 치주 상태, 충치 여부, 치석 정도, 생활습관 등)  
- 데이터 전처리 및 특징 추출  
- 다양한 머신러닝 모델 비교 및 최적 모델 선정  
- 시각화를 통한 결과 해석

## 6. 기대 효과
- 환자 맞춤형 치료 및 예방 전략 수립 가능  
- 조기 진단을 통한 치료 비용 절감  
- 치의학 데이터 기반 의사결정 강화
